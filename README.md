# Yelp-Restaurants-Clustering
<h1>Mapping Bay Area restaurants on Yelp</h1>


<h2>Description</h2>
In this class porject, I performed k-means clustering to categorize Berkeley restaurants into clusters based on geographic proximity, by iteratively calculating optimal centroids through averaging geographic coordinates. Thereafter, I implemented least-squares regression to predict user ratings and applied r-squared analysis to optimize features. Finally, I was able to map alike restaurants via a Voronoi diagram color-coded visualization.
